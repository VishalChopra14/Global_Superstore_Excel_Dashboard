# 📊 Global Superstore Excel Dashboard

A fully interactive **Excel Dashboard** created using the Global Superstore dataset to analyze Sales, Profit, Discounts, Categories, Segments, Shipping Modes, and Regional performance.  
This project highlights strong Excel analytics and dashboard design skills using PivotTables, KPI cards, slicers, and advanced formulas.

---

## 🖼 Dashboard Preview

![Dashboard Preview](https://github.com/VishalChopra14/Global_Superstore_Excel_Dashboard/blob/main/global%20superstore%20dashboard.png)

---

## 🚀 Key Features

- **Dynamic KPI Cards**
  - Total Sales, Profit, and Discount update automatically based on slicers.
- **Interactive Slicers**
  - Segment, Region, Country, Category, Sub-Category, Ship Mode, Order Priority, and Date.
- **PivotTables & PivotCharts**
  - For advanced analysis and drill-down capabilities.
- **Clean & Professional UI**
  - Consistent theme, icons, and layout.
- **GETPIVOTDATA-based KPIs**
  - Ensures KPI values stay linked to filtered Pivot data.
